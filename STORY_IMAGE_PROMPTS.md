# Story Mode Chapter Image Prompts

This document contains detailed prompts for generating images for each story mode chapter. Use these with AI image generators like Midjourney, DALL-E 3, or Stable Diffusion.

## Image Specifications
- **Aspect Ratio**: 16:9 (landscape) or 4:3
- **Resolution**: At least 1920x1080 for best quality
- **Style**: Warm, welcoming, realistic with slight artistic touch
- **Color Palette**: Earthy tones, warm lighting, inviting atmosphere
- **File Format**: PNG or JPG
- **Filename Convention**: `chapter-{number}-{slug}.png`

## How to Use These Prompts

### With Midjourney:
```
/imagine [paste prompt here] --ar 16:9 --v 6 --style raw
```

### With DALL-E 3:
Simply paste the prompt and specify landscape orientation.

### With Stable Diffusion:
Use the detailed prompt with these settings:
- Steps: 30-50
- Sampler: DPM++ 2M Karras
- CFG Scale: 7-9
- Model: Realistic Vision or DreamShaper

### With Leonardo.AI:
- Select "PhotoReal" mode
- Paste the prompt
- Set aspect ratio to 16:9
- Elements: Photography, Pet Photography, Professional

## Tips for Best Results

1. **Consistency**: Use similar lighting and color grading across all chapters
2. **Quality**: Generate at highest resolution available
3. **Variations**: Generate 2-3 variations and pick the best
4. **Editing**: Use basic photo editing to adjust brightness/contrast if needed
5. **Testing**: View images in the app to ensure they work well with the UI
